# Horizon Taxonomy

时间周期不是修饰词，而是研究方向、执行市场、风险预算和退出规则的主轴。

## 短线分层

### `h24-48`

- 典型含义：今天到后天
- 适用：
  - 美股/商品/加密的快速信号交易
  - 美股先行、A股/港股跟随
- 重点：
  - 催化强度
  - lead-lag
  - 执行可达性
  - 拥挤度过滤

### `d3-7`

- 典型含义：3天到一周
- 适用：
  - A股短线主流
  - 港股短 swing
- 重点：
  - 资金持续性
  - 事件扩散
  - 板块共振

### `w1-2`

- 典型含义：一周到半个月
- 适用：
  - 主题发酵
  - 预告 / 政策 / 产品周期前移
- 重点：
  - 二阶催化
  - 预期差
  - 板块轮动

### `w2-4`

- 典型含义：半个月到一个月
- 适用：
  - 波段交易
  - 业绩前 / 后修正
- 重点：
  - 估值和资金共振
  - 事件兑现节奏

## 长线分层

### `m1-3`

- 适用：
  - 中短期长线
  - 行业和业绩趋势交易

### `m3-6`

- 适用：
  - 宏观 / 商品 / 产业链景气交易

### `m6-12`

- 适用：
  - 年内主线
  - 资本开支、供需、盈利兑现

### `y1-3`

- 适用：
  - 中期复利
  - 竞争格局改善
  - 资源品上行周期

### `y3-5`

- 适用：
  - 结构性成长
  - 龙头份额扩张
  - 周期与结构双重受益

### `y10-plus`

- 适用：
  - 超长期战略配置
  - 复利型资产
  - 大宗资源与硬资产长期供需错配

## 默认推断规则

用户未指定时：

### Short

- `market=cn` -> 默认 `d3-7`
- `market=hk` -> 默认 `d3-7`
- `market=us` -> 默认 `h24-48`
- `market=futures|crypto|options` -> 默认 `h24-48`
- `market=multi + alpha_mode=lead-follow` -> 默认 `h24-48`

### Long

- `instrument=equity|fund|convertible_bond` -> 默认 `m1-3`
- `instrument=commodity|index + macro thesis` -> 默认 `m3-6`
- 若用户明显表达“长期持有 / 复利 / 战略看多” -> 默认至少 `m6-12`，再往上评估

## 报告必须写清楚

- 选择了哪个周期层
- 为什么不是更短或更长
- 进入窗口
- 退出窗口
- 最大持有时长
- 特殊情况怎么提前退出或延长持有
