# Interpersonal Relations Module

用于对上管理、平级冲突、高管协作、团队信任、个人压力与影响力问题。

## 适用信号

- 老板预期不清、平级掣肘、团队关系紧张
- 高管协作不顺、信任下降、会议气氛失控
- 用户情绪负荷高，需要先稳住局势再推进

## 核心看点

- 冲突类型：任务冲突、关系冲突、控制权冲突
- 关键人地图：赞助者、阻击者、摇摆者、围观者
- 情绪温度：焦虑、愤怒、羞耻、委屈、防御
- 面子与边界：哪些能公开说，哪些必须私下说
- 影响力来源：职位、专业、信息、联盟、节奏

## 默认问题

1. 这件事是道理问题，还是位置问题
2. 谁真正害怕失去什么
3. 这次沟通的目标是说服、止损、设边界，还是重建信任
4. 如果必须留体面，台阶怎么搭

## 默认输出

- 关系局势判断
- 关键人立场图
- 对话和会议设计
- 体面转弯方案
- 团队稳定动作

## 常见协作模块

- 涉及高管去留与控制权 -> `executive-personnel-conflict.md`
- 涉及绩效、保留和继任 -> `performance-optimization.md`
- 涉及跨文化团队 -> `cross-cultural-management.md`

## 易错点

- 把组织政治简化成“谁对谁错”
- 该私下说的事放到会上说
- 只讲道理，不给对方安全感
