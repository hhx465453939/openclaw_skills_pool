# Executive Consultant Playbooks

以下打法用于把判断变成动作。可以组合使用，但不要同时抛出过多框架。

## 使用原则

- 默认先用低摩擦、可执行、可回退的管理打法
- 只有当控制权、合规、重大风险或高压时限明确存在时，才升级到强势动作
- 先区分“普通管理问题”和“复杂局面问题”，不要把所有问题都按斗争处理
- 每次建议都尽量给一个普通管理版和一个复杂局面增强版

## 1. 人事处置与留人

适用：招人、留人、优化、降级、换岗、继任、绩效纠偏。

默认姿态：

- 普通管理问题：先纠偏、再决定是否换岗或退出
- 复杂局面问题：当信任断裂、关键岗位失控或风险上升时，再进入强处理

默认步骤：

1. 分清问题是能力、意愿、信任、价值观还是岗位错配
2. 用事实和记录支撑动作，不要靠情绪印象
3. 先明确终局：要挽回、纠偏、平稳退出，还是立即切割
4. 先和必要的老板、HR、法务私下对齐，再谈当事人
5. 一次谈清决定、时间表、支持措施和留痕方式

话术骨架：

- 普通管理版：“这次先把问题和预期说清楚，我们先看纠偏窗口能不能成立。”
- “今天先把方向说清楚：这件事已经进入调整阶段，不再停留在提醒阶段。”
- “这不是对你个人价值的否定，但当前岗位和组织需要已经不匹配。”
- “接下来两周我们只做三件事：交接、书面确认、后续安排。”

易错点：

- 一边想留人，一边用羞辱式语言
- 证据不足却想做重动作
- 给出无法兑现的模糊承诺

## 2. 谈判与合作推进

适用：商务谈判、股东协调、薪酬谈判、合作续签、并购条件交换。

默认姿态：

- 普通场景：优先用交换、澄清和节奏控制，不先压迫
- 高压场景：只有当窗口期短、失误代价大时，才提高锚点和止损强度

默认步骤：

1. 先写清 BATNA、止损线、理想结果和最差可接受结果
2. 先探盘，再设锚点，不要一上来把底线说透
3. 只做交换，不做裸让步
4. 重要条款打包谈，不让对方逐条拆穿
5. 收口时复述共识、未决项和截止时间

话术骨架：

- 普通管理版：“这个条件我们可以继续谈，但需要一起看整体平衡。”
- “这个点可以谈，但必须和 X 一起打包。”
- “如果要这个条件，我们需要等价对价。”
- “我们的底线不是情绪，是基于资源和风险测算出来的。”

易错点：

- 用假数据、假授权、假 BATNA 吓人
- 只盯价格，不设计非价格交换项
- 没有明确 walk-away 条件

## 3. 对上管理与组织政治

适用：争授权、保位置、处理平级掣肘、被架空、抢资源、敏感会议。

默认姿态：

- 普通管理问题：先对齐目标和责任，不预设对抗
- 复杂局面问题：当存在越权、架空、定义权争夺时，再按政治问题处理

默认步骤：

1. 先画出四类人：赞助者、阻击者、摇摆者、围观者
2. 关键分歧优先私下对齐，不在会上做 surprise
3. 让老板感到安全，让平级有台阶，让团队知道边界
4. 对资源、结果、责任三件事分别留痕
5. 必要时给对方留体面转弯空间，但不牺牲核心控制权

话术骨架：

- 普通管理版：“我想先确认目标一致，再把责任和边界写清。”
- “我想先和您单独对齐，不希望在会上制造额外噪音。”
- “我不反对目标，我反对当前路径和授权设计。”
- “如果由我承担结果，就需要同步明确资源和边界。”

易错点：

- 当众打脸，逼对方公开站队
- 迷信‘讲道理就够了’，忽视位置和安全感
- 靠传话、暗示和流言代替正式动作

## 4. 政商、外资与跨文化协同

适用：总部与本地、外资与中方、政府关系、合资方、跨文化团队。

默认姿态：

- 普通场景：先做桥接和翻译，不先做压强和站队
- 复杂局面：只有在权力、审批、面子和风险同时存在时，再叠加专项模块

默认步骤：

1. 把原则问题、执行问题、面子问题拆开看
2. 分清正式口径、内部口径、对外口径
3. 重大事项尽量准备一页纸摘要，避免多轮口头误解
4. 安排沟通顺序时，先保关键关系，再推具体动作
5. 用留面子的方法推进，不用模糊承诺换短期顺利

话术骨架：

- “原则上我们支持这个方向，执行上需要按 X 个步骤推进。”
- “这个版本适合内部对齐，公开口径建议另行整理。”
- “我们可以加速，但不会绕开必要的审批和记录。”

易错点：

- 误把礼貌当承诺，或把沉默当同意
- 试图用‘口头默契’替代合规流程
- 把文化差异解释成能力差或忠诚问题

## 5. 危机、合规与举报

适用：举报、调查、舆情、事故、第三方违规、敏感人事事件。

默认姿态：

- 这一类默认偏稳，不追求表达锋利，优先事实、边界和留痕

默认步骤：

1. 先保全事实、证据、系统和关键人员
2. 立刻收窄发声口，避免多人自由发挥
3. 区分已核事实、待核事实、假设和情绪判断
4. 识别是否需要律师、合规、审计、HR 或对外顾问介入
5. 对外口径只说已核实信息，不抢结论

话术骨架：

- “现在先不判断责任，先锁定事实和证据。”
- “对外只说已核实部分，不做推测。”
- “任何材料先保全，不删不改，不做事后修饰。”

易错点：

- 先想甩锅，再想取证
- 私下授意统一口径，实质上变成串供
- 把举报处理和对举报人的绩效处理混在一起

## 6. 变革、重组与文化修复

适用：组织调整、流程重构、文化重塑、业务转型、并购后整合。

默认姿态：

- 普通场景：先试点、先 quick wins、先补承接能力
- 复杂局面：只有在时间窗口极紧或组织失控时，才考虑强推

默认步骤：

1. 先定义为什么现在必须变，不变的代价是什么
2. 识别谁会赢、谁会输、谁会阳奉阴违
3. 第一阶段只动最关键的三件事，先拿 quick win
4. 同步重写激励、汇报线和节奏，不只发口号
5. 给中层管理者额外支持，因为他们最容易成为阻力放大器

话术骨架：

- 普通管理版：“这次先从最关键的两三项开始，不一次性把所有东西推倒重来。”
- “这次调整不是为了表态，而是要解决 X。”
- “短期会不舒服，但如果不改，代价会落在 Y 上。”
- “第一阶段只动三件事，其余事项先冻结。”

易错点：

- 讲太多愿景，讲太少损失和交换
- 只要求员工配合，不调整授权和激励
- 低估中层的消极抵抗
