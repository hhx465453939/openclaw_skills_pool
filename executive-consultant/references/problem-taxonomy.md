# Executive Consultant Problem Taxonomy

先选一级问题域，再选子类，再决定主模块和协作模块。一级问题域的判断标准只有一个：哪一类问题如果处理错，总代价最大。

## 1. `STRATEGY` 战略决策类

适用：方向调整、增长路径、资本与股东、并购整合、重大合作与危机方向判断。

常见子类：

- `MA`：并购重组、拆分、整合、退出
- `TRANSFORMATION`：数字化、业务转型、模式重构
- `GROWTH`：新市场、新产品、新合作、生态布局
- `CRISIS`：重大经营、法律、品牌或舆情危机中的方向选择

主模块：

- 主模块：战略诊断
- 常见协作：组织效能、变革管理、跨文化管理

## 2. `ORGANIZATION` 组织效能类

适用：组织架构、人才管理、文化落地、绩效机制、跨部门协作。

常见子类：

- `STRUCTURE`：组织架构、汇报线、边界模糊
- `TALENT`：关键人才、梯队建设、岗位匹配
- `CULTURE`：文化重塑、价值观落地、士气与氛围
- `PERFORMANCE`：绩效体系、激励机制、结果与责任对齐

主模块：

- 主模块：组织效能
- 常见协作：绩效优化、变革管理、人际关系

## 3. `LEADERSHIP` 领导力发展类

适用：对上管理、团队领导、冲突管理、影响力建设、政商与投资者关系。

常见子类：

- `PERSONAL`：个人领导风格、决策风格、压力管理
- `TEAM`：团队领导力、高管协作、冲突与赋能
- `CHANGE`：变革领导力、变革沟通、抵触应对
- `RELATIONS`：政府关系、媒体关系、投资者关系、关键外部影响力

主模块：

- 主模块：人际关系
- 常见协作：变革管理、跨文化管理、绩效优化

## 4. `OPERATIONS` 运营优化类

适用：流程优化、成本管理、质量改进、创新机制、执行节奏。

常见子类：

- `PROCESS`：流程再造、审批链、决策路径
- `COST`：成本结构、资源配置、效率提升
- `QUALITY`：质量体系、客户满意度、品牌声誉
- `INNOVATION`：创新机制、敏捷试点、组织学习

主模块：

- 主模块：组织效能
- 常见协作：战略诊断、绩效优化

## 5. `RISK` 风险管控类

适用：合规、举报调查、声誉风险、经营连续性、劳动与财务风险。

常见子类：

- `STRATEGIC_RISK`：战略误判、竞争风险、技术颠覆
- `OPERATIONAL_RISK`：合规、供应链、信息安全、内部控制
- `FINANCIAL_RISK`：资金链、预算失控、舞弊、汇率
- `REPUTATION_RISK`：舆情、品牌危机、领导者个人声誉

主模块：

- 主模块：战略诊断
- 常见协作：人际关系、变革管理、`escalation-boundaries.md`

## 高频问题映射

以下高频问题应优先命中：

- 团队激励与士气管理 -> `ORGANIZATION/PERFORMANCE` -> 绩效优化 + 人际关系
- 战略决策风险分析 -> `STRATEGY/CRISIS` -> 战略诊断
- 跨部门冲突调解 -> `LEADERSHIP/TEAM` -> 人际关系 + 组织效能
- 人才保留与离职预防 -> `ORGANIZATION/TALENT` -> 绩效优化
- 绩效体系设计优化 -> `ORGANIZATION/PERFORMANCE` -> 绩效优化 + 组织效能
- 文化变革推动 -> `ORGANIZATION/CULTURE` -> 变革管理 + 组织效能
- 危机公关与声誉管理 -> `RISK/REPUTATION_RISK` -> 战略诊断 + 人际关系
- 跨文化团队融合 -> `LEADERSHIP/TEAM` 或 `ORGANIZATION/CULTURE` -> 跨文化管理
- 政策合规风险管理 -> `RISK/OPERATIONAL_RISK` -> 战略诊断 + `escalation-boundaries.md`
- 继任计划与人才梯队 -> `ORGANIZATION/TALENT` -> 绩效优化
- 数字化转型战略 -> `STRATEGY/TRANSFORMATION` -> 战略诊断 + 变革管理
- 组织架构调整 -> `ORGANIZATION/STRUCTURE` -> 组织效能
- 高管团队协作优化 -> `LEADERSHIP/TEAM` -> 人际关系 + 组织效能
- 压力管理与心理健康边界 -> `LEADERSHIP/PERSONAL` -> 人际关系
- 知识管理与学习型组织 -> `OPERATIONS/INNOVATION` -> 组织效能
- 创新驱动与变革阻力 -> `OPERATIONS/INNOVATION` 或 `LEADERSHIP/CHANGE` -> 变革管理
- 并购后文化整合 -> `STRATEGY/MA` -> 跨文化管理 + 变革管理
- 道德困境与伦理决策 -> `RISK/OPERATIONAL_RISK` -> 战略诊断 + `escalation-boundaries.md`
- 政商关系维护策略 -> `LEADERSHIP/RELATIONS` -> 人际关系 + `china-gov-business.md`
- 股东关系管理 -> `STRATEGY/GROWTH` 或 `LEADERSHIP/RELATIONS` -> 战略诊断

## 默认优先级

当问题同时涉及多条线，按以下顺序排序：

1. 人身安全、证据保全、法定义务
2. 方向错误与控制权风险
3. 业务连续性与关键关系
4. 组织稳定、士气和文化后果
5. 中长期效率、创新和形象优化
